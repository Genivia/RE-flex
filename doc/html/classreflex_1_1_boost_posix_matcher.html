Detailed Description

Boost matcher engine class, extends reflex::BoostMatcher for Boost POSIX regex matching.

Boost POSIX regex matching enables Boost match flags match_posix and match_not_dot_newline. Lazy quantifiers are not supported by this matcher engine.
